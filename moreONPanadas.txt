Axis - 

https://www.youtube.com/watch?v=PtO3t6ynH-8

axis = 0: all rows for a specific column or all rows of all columns as define
for example: df.drop(2,axis=0) meaning droping all columns of row 2
df.mean(axis=0) ----mean/avg of all rows of each column

df.drop('conuntry',axis =1) means drop all rows of column name country

df.mean(axis=1)--- mean/avg of all columns of each row 

axis =0 move across all left to right going down by each column ---
axis = 0 can be also written as axis='index'

axis 1 = move top to down going across (left to right) all columns 
axis =1 can be also written as axis='columns'
